# ⚡ ReVinted: AI Resale Intelligence

![ReVinted Banner](https://images.unsplash.com/photo-1597040966242-488c783a7403?auto=format&fit=crop&q=80&w=1200&h=400)

**ReVinted** è la piattaforma definitiva per reseller professionisti su Vinted, Depop ed eBay. Utilizza l'intelligenza artificiale avanzata per trasformare semplici foto in analisi di mercato profonde, stime di prezzo precise e annunci pronti per la vendita.

---

## ✨ Caratteristiche Principali

- **🔍 Analisi Visiva GPT-4o**: Identificazione istantanea di brand, modelli, epoche e materiali partendo dalle foto.
- **💰 Stime di Prezzo Dinamiche**: Ricevi una stima del valore attuale basata sui dati di mercato in tempo reale (Vinted, eBay).
- **📉 Proiezioni Future**: Scopri se il tuo oggetto guadagnerà valore nel tempo grazie alle previsioni a 1, 3 e 5 anni.
- **🎯 Market Scout**: Un motore di ricerca "Live" che scansiona i marketplace alla ricerca di affari (steal) con alto potenziale di profitto.
- **📝 Multi-Platform Listing**: Genera automaticamente titoli SEO e descrizioni ottimizzate per Vinted, Depop ed eBay con un click.
- **📈 Dashboard Vendite**: Traccia il tuo portfolio, i tuoi guadagni reali e il ROI medio dei tuoi "flip".

---

## 🎨 Design System: Neo-Brutalist

ReVinted non è solo potente, è anche bello da vedere. Abbiamo scelto un'estetica **Neo-Brutalist** che comunica forza, trasparenza e modernità:
- Bordi neri spessi (4px)
- Ombre proiettate nette (Shadow Pop)
- Palette colori vibrante (Vinted Teal, Vibrant Yellow, High-Contrast Black)
- Micro-animazioni fluide per un feedback istantaneo

---

## 📱 Mobile First & PWA

Ottimizzato specificamente per **iPhone 15**, ReVinted offre un'esperienza mobile nativa grazie alla sua architettura **PWA (Progressive Web App)**:
- Navigation Bar inferiore per un uso immediato con una sola mano.
- Supporto Safe-Area per notch e Dynamic Island.
- Velocità di caricamento estrema e feedback aptico visivo.

---

## 🛠️ Tech Stack

- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Supabase (Auth, Database, Edge Functions)
- **AI**: OpenAI GPT-4o & GPT-4o-vision
- **Charts**: Recharts per visualizzazione dati finanziari
- **Icons**: Lucide React
- **Animations**: Framer Motion & Tailwind Transitions

## 🚀 Come Iniziare

1. **Clona il repository**
   ```bash
   git clone https://github.com/your-username/vinted-valuator.git
   ```
2. **Installa le dipendenze**
   ```bash
   npm install
   ```
3. **Configura le variabili d'ambiente**
   Crea un file `.env` con le tue chiavi Supabase:
   ```env
   VITE_SUPABASE_URL=your_url
   VITE_SUPABASE_ANON_KEY=your_key
   ```
4. **Avvia il server di sviluppo**
   ```bash
   npm run dev
   ```
